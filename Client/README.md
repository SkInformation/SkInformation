## How to Run
### Step 1
Install [Node.js](https://nodejs.org/en)

### Step 2
Install the dependencies
```bash
npm i
```

### Step 3
Run the project
```bash
npm run dev
```

### Step 4
Open your browser to [http://localhost:3000](http://localhost:3000)
